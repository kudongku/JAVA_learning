# 연산자와 조건문, 반복문
## 연산자 종류
- 산술 연산자 1+2
- 비교 연산자 >=
- 논리 연산자 ||
- 대입 연산자 ++
- 형변환 연산자 (int)
- 삼항 연산자
  ```
    return (a==b)? "진실":"거짓";
  ```
- instance of 연산자
- 바트 연산자

## 조건문
- if, else if, else
- switch
- 향상된 switch

## 반복문
- for 
- 향상된 for
  ```
  for(int i : intArr){
  }
   ```
  단, i 에 관하여 수정은 불가
- while
- do while
- break
- continue
